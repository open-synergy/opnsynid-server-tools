===================
Base Public Holiday
===================

This module is a combination of module "hr_public_holidays" belongs to OCA(https://github.com/OCA/hr)
and "hr_public_holidays_ics_import" belongs to IT-Projects LLC(https://it-projects.info).

Installation
============

To install this module, you need to:

1.  Clone the branch 8.0 of the repository https://github.com/open-synergy/opnsynid-server-tools
2.  Add the path to this repository in your configuration (addons-path)
3.  Update the module list
4.  Go to menu *Setting -> Modules -> Local Modules*
5.  Search For *Base Public Holiday*
6.  Install the module
